Adding Content on Oxygen

In this step, you'll learn how to start adding content to Oxygen

This task includes learning how to create a ditamap and create files, implement titles, descriptions, subheadings, paragraphs, and more

Steps

  1. Creating a DITA map
    1. Click on the three dots to open files
    2. Click "New File"
    3. Click on the tab marked "DITA" on the left side menu.
    4. Click on the option titled "Map".
    5. Name the .ditamap "document".
    6. Click the blue button labeled "OK".
  2. Creating a Topic
    1. Click on the three dots to open files
    2. Click "New File"
    1. Click on the tab marked "DITA" on the left side menu.
    1. Click on the option titled "Topic".
    2. Give the New File a name
    3. Click the blue button labeled "OK".
  3. Drag the topic file into the DITA map file
    1. Start adding your content
      1. Click the "Return" key on your keyboard anywhere on the Topic File.
      2. Type in the text box whatever type of content you would like to add to that Topic File. Examples could be Titles, Subtitles, Short descriptions, Sections, Figures, Images, Adding alt text, Paragraphs, etc.
