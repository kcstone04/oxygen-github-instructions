Making a Project in Oxygen Content Fusion

In this step, you will learn how to create a project on Oxygen Content Fusion.

This task includes learning how to create a project file on Oxygen Content Fusion that links to the repository you created in GitHub.
  1. Hover over the top left corner of the screen and select the "Projects" tab
    1. Click "New Project" in Oxygen Content Fusion
    1. Create a name for your new project
    1. Copy the link from your repository
    2. Paste link of your repository into Oxygen
    1. Set the branch to "master"
    1. Paste your GitHub Username into Oxygen
    1. Get a GitHub token to connect it to the GitHub repository
      1. Click the link next “Generate Access Token”
      2. Click “Generate New Token” at the top left hand of the screen.
      3. Type in a new name for this token
      4. Click on “Only Selected Repositories”
      5. Select the repository you made that you would like to connect to Oxygen.
      6. Click “Repository Permissions”
      7. Give it Read and Write Access under Contents
      8. Click Generate Token
      9. Copy Access Token
      10. Paste Access Token into Oxygen
    2. Click Create Project at the bottom of the screen.
