Creating the GitHub Repository

In this step, you will learn how to create a repository in GitHub.

This step will teach you how to create the repository based off of the GitHub Actions template. This is necessary to get the GitHub side of your project truly started.

  1. Open up the GitHub Actions template.

    Template link: https://github.com/VT-Evia/dita-ot-actions

  2. Click the button that says "Use This Template" in the top right of the screen, then select "Create Repository".
  3. Select "Include all Branches".
  4. Name your new repository.
  5. Click "Create Repository" to finalize it.
  6. Once the repository is made, Click "Settings"
    1. Click "Actions"
    1. Scroll down to"Workflow Permissions"
    1. Click "Read and write permissions".
