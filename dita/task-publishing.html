Publishing to GitHub Using GitHub Actions

In this step, you’ll learn how to publish your content.

This step will walk you through the process of publishing your content through GitHub Actions. This is the point where your work in Oxygen and GitHub truly merges into one single workflow.

  1. Make Sure You Have a .ditamap File in Oxygen.
    This is necessary, as this is the file that GitHub will be trying to pull from. Name it "document.ditamap" to make sure it works.
  2. Ensure that all changes made on Oxygen are committed.
    1. From the Oxygen personal workspace, Click "Changes" on the left sidebar.
    2. Press the blue "Commit" button.
    3. Write a message briefly describing what you have changed.
    4. Click the blue "Commit Button".
  3. Finding your GitHub PDF deliverables!
    1. Go to your Repositories. Found at the top menu bar.
    2. Click on the repository you would like to publish.
    3. Click on "Actions".
    4. Click on your most recent commit.
    5. Click on the link of the PDF file at the bottom of the page.
  4. Finding your GitHub HTML deliverables!
    1. Go to your "Repositories" Found at the top menu bar.
    2. Click on the repository you would like to publish.
    3. Click "Settings".
    4. Click "Pages".
    5. Click the link to the website at the top of the page.
